## 目录结构说明

```sh
├─doc # 一些说明文档
│  └─sql # 数据库初始化脚本
├─router # 路由
│  ├─...
├─static # 静态资源
│  └─upload # 上传文件服务存储位置
├─test # 测试文件（.gitignore忽略）
├─utils # 工具类
│  ├─db.js # 数据库连接池
│  └─Response.js # 请求响应处理对象
├─index.js  # 主接口服务器
├─uploadServer.js  # 上传文件服务器
├─chatServer.js  # 聊天服务器
├─.env  # 配置

```
